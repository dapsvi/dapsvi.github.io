Instructions for Cardputer :

  • Option 1 : website flasher. Head over to bruce.computer/flasher, and follow the instructions. You will have to turn your cardputer off, unplug the USB, hold the G0 button and replug it. Select the version of the firmware wanted and then select M5Stack then Cardputer. You can now flash the device. Make sure you are using a chromium-based browser.
  • Option 2 : M5Burner. Download the official M5Burner application, and select the Cardputer. From there, search "Bruce", download it and burn it.

Instructions for M5StickC :

  • Option 1 : website flasher. Head over to bruce.computer/flasher, and follow the instructions. Select M5Stack then M5Stick. You can now flash the device. Make sure you are using a chromium-based browser.
  • Option 2 : M5Burner. Download the official M5Burner application, and select the M5Stick. From there, search "Bruce", download it and burn it.
